# Photo Album

A simple, browser-based photo album app built with vanilla HTML, CSS, and JavaScript.

## Features

- Add photos via image URL
- View photos in a responsive grid layout
- Click any photo to open a full-size lightbox
- Delete photos with the hover delete button
- Photos are persisted in `localStorage`

## Usage

Open `index.html` in any modern browser — no build step or server required.
